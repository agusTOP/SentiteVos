# Sentite Vos - Estética y Bienestar

## Estructura del Proyecto

```
ProyectoFinal/
├── index.html
├── contacto.html
├── galeria.html
├── servicios.html
├── testimonios.html
├── styles/
│   └── styles.css
├── scripts/
│   └── main.js
├── assets/
│   └── images/
│       └── (aquí van tus imágenes propias o libres)
└── README.md
```

## ¿Cómo editar?
- **index.html**: Página principal, aquí se integran todas las secciones.
- **styles/styles.css**: Todos los estilos y personalización visual.
- **scripts/main.js**: Código JavaScript para interactividad.
- **assets/images/**: Guarda aquí tus imágenes, fotos del equipo, etc.
- **favicon**: Puedes cambiar el favicon editando la línea correspondiente en el `<head>` de `index.html`.

## Créditos
- Imágenes de Unsplash (puedes reemplazarlas por propias).
- Iconos SVG minimalistas integrados en el HTML.

## Notas
- El proyecto es 100% HTML, CSS y JS puro, sin frameworks.
- Estructura lista para crecer y mantener fácilmente.
